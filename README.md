## 一键绕过付费墙，免费阅读全球优质新闻

一个简单易用的浏览器插件，让你通过 Archive.today 免费阅读各大付费新闻网站的完整内容。无需订阅，无需登录，一键直达。

---

## ✨ 主要功能

🎯 **一键跳转** - 页面右下角悬浮按钮，点击即可跳转到 Archive 存档版本  
🌍 **支持50+网站** - 覆盖全球主流付费新闻、财经、科技媒体  
🎨 **优雅设计** - 简洁美观的黑色悬浮按钮，不干扰阅读  
⚡ **快捷键支持** - 按 `Alt + A` 或 `Ctrl/Cmd + Shift + A` 快速跳转  
🧹 **智能URL清理** - 自动移除跟踪参数，确保能找到存档  
🎭 **轻量无痕** - 不收集数据，不修改页面内容  
🔄 **实时更新** - 始终访问最新的存档版本

## 📖 使用指南

### 方法一：点击按钮
访问付费文章页面时，会在右下角看到一个黑色悬浮按钮：

```
┌─────────────────┐
│ 📖 Read Free    │
│    (Archive)    │
└─────────────────┘
```

点击按钮，即可在新标签页打开该文章的 Archive 存档版本。

### 方法二：快捷键
- **通用版本**: 按 `Alt + A`
- **智能版本**: 按 `Ctrl + Shift + A` (Windows) 或 `Cmd + Shift + A` (Mac)

### 方法三：拖动按钮（仅智能版）
按住鼠标左键可以拖动按钮到任意位置，方便不同阅读习惯。

## 🔍 工作原理

1. **检测付费网站** - 脚本识别你访问的网站是否在支持列表中
2. **清理URL** - 移除 `?mod=xxx` 等跟踪参数，获得干净的链接
3. **跳转Archive** - 将清理后的URL发送到 `archive.today/latest/[URL]`
4. **展示内容** - Archive.today 显示该网页的完整存档版本

**为什么有效？**
- Archive.today 会定期抓取并保存网页快照
- 存档版本不包含付费墙的JavaScript限制
- 内容完整，包括图片、评论等

## 💡 常见问题

### Q1: 为什么有些文章找不到存档？
**A:** 可能的原因：
- 文章太新，Archive 还未抓取（通常24小时内会有）
- 文章URL包含特殊参数，尝试手动访问 archive.today 搜索
- 该网站使用动态加载，Archive 无法完整保存

**解决方法:**
- 等待几小时后再试
- 手动去 [archive.today](https://archive.today) 提交该链接
- 尝试其他存档服务如 [web.archive.org](https://web.archive.org)

---

### Q2: 这合法吗？
**A:** Archive.today 是公开的网页存档服务，类似于图书馆的剪报服务。但请注意：
- ⚖️ 法律因国家/地区而异
- 📄 仅供个人学习研究使用
- 💰 如果经济允许，建议支持原创作者，购买正版订阅
- 🎓 学生/研究者可申请机构访问权限

---

### Q3: 为什么按钮不显示？
**A:** 检查以下几点：
1. 确认 Tampermonkey 已启用
2. 确认脚本已安装并启用
3. 刷新页面（Ctrl+F5 强制刷新）
4. 检查网站域名是否在支持列表中
5. 查看浏览器控制台是否有报错

---

### Q4: 可以添加更多网站吗？
**A:** 当然可以！编辑脚本，添加新的匹配规则：

**通用版:**
```javascript
// @match https://新网站.com/*
```

**智能版:**
在 `paywallSites` 数组中添加域名：
```javascript
const paywallSites = [
    // ... 现有网站
    '新网站.com'
];
```

---

### Q5: 存档版本有广告吗？
**A:** Archive.today 本身包含少量广告，但没有付费墙的弹窗和限制，阅读体验更好。

---

### Q6: 能保存图片和视频吗？
**A:** 
- ✅ 静态图片：通常能完整保存
- ❌ 视频：大多数视频无法保存
- ❌ 动态内容：JavaScript生成的内容可能缺失
